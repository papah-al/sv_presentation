Billing

Primary Task

Billing Checking & Creation (Max. Potential vs Accrual / Actual Comparison).
Before Condition
Submission direct from UM / GM to Sales Support.
There is no calculation of billing performance.
Current Condition
Submission from UM / GM to WFM for checking and creation and submitted to Sales Support.
Create calculation of billing performance (Max. Potential vs Accrual / Actual Comparison).
Next Plan
Can use data output from Trasforce or Excel template for each project on a daily basis.
Goals / Outcomes
Efficient processing in line with the sales support timeline, with more optimal and accurate billing.
Centralized Billing Checking & Creation by WFM for all project.
Developing a tool that makes it easier for management to access and analyze billing performance.

Profit & Loss (PL)

Primary Task

Checking Budget vs Actual PL Monthly and create gap analysis.
Before Condition
Dashboard tool provide by Finance Division.
Delayed changed if any update of organizational structure during the current year.
Current Condition
Raw data from Finance Division, dashboard tool provide by WFM (custom with UM / GM visibility).
Adaptive with changes in organizational structure.
Developing additional template for summary gap analysis (Budget vs Forcast vs Actual).
Next Plan
Transform to web based (Transforce - PL Dashboard & Analysis).
Goals / Outcomes
Developing a tool that makes it easier for user and management to access, analyze and make forcast / target PL.
